# Stationeers Screens Designer

A dedicated design tool to simplify and accelerate the creation of custom UIs for *Stationeers*, specifically built for the [ScriptedScreens Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=3666779631).

---

## ⚠️ Important Warning
**This is an early-access/alpha release.**
* **Bugs:** Expect bugs.
* **No Autosave:** This software does not currently feature an autosave system. **Please save your work frequently.**
* **Compatibility:** This software is provided "as-is" without any guarantees. As this is an external tool, there is no guarantee that generated UIs will be compatible with future updates to the *ScriptedScreens Mod* or *Stationeers* itself.
* **Major Changes:** The software and its file formats are subject to significant changes in future updates.

---

## ⌨️ Shortcuts

Master the interface with these keyboard shortcuts:

| Action | Shortcut |
| :--- | :--- |
| **Delete** | `Del` |
| **Duplicate** | `Ctrl + D` |
| **Multi-select** | `Ctrl + Click` |
| **Rename (Tab)** | `F2` (hover over tab) |
| **Snap to Grid** | `Shift + Drag` |
| **Zoom** | `Ctrl + Scroll` |
| **Horizontal Scroll** | `Ctrl + Shift` |
| **Vertical Scroll** | `Scroll` |

> **Note:** The Lua Code viewer supports standard editing shortcuts like `Ctrl + C` (Copy) and `Ctrl + A` (Select All).

---

## 🛠 Prerequisites
This tool is designed to work in conjunction with the [ScriptedScreens Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=3666779631). Please ensure you have the mod installed and configured in *Stationeers* to use the output generated by this designer.

This tool runs on .NET 10.0 (Windows) and probably won't appear for other operating systems anytime soon. [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
